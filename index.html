📁 GitHub Pages – step‑by‑step (no prior knowledge)

  1. Create repository: Log in to GitHub, click the green "New" button. Name it e.g. fuel-converter, keep public, click "Create repository".
  2. Upload index.html: On the new repo page, click "Add file" → "Upload files". Drag and drop this index.html (save the code above as a file named index.html). Then click "Commit changes".
  3. Enable Pages: Go to repository "Settings" → "Pages" (left sidebar). Under "Branch", select "main" and "/ (root)", then "Save".
  4. Get your URL: After a minute, refresh the Pages section – you'll see a message like "Your site is live at https://yourusername.github.io/fuel-converter".
  5. Done: Anyone can now use the tool via that link. Updates: just upload a new index.html to the same repository.

✅ No API keys needed – exchange rates from European Central Bank (frankfurter.dev). All conversion logic runs in your browser.
